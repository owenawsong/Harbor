# Changelog

All notable changes to Harbor are documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2025-03-22

### Added
- Initial public release of Harbor AI Extension
- AI-powered code completion and suggestions
- Multiple AI provider support (Harbor, Claude, OpenAI)
- Keyboard accessibility improvements
- Quick setup wizard for initial configuration
- Skills gallery for discovering AI capabilities
- Smooth animations and enhanced UI/UX
- Agent tab with indicator and command palette integration
- Content script integration for enhanced functionality
- Dashboard with AI interactions overview
- Chat interface with multi-turn conversations
- Settings panel for configuration management
- Persistent state management
- TypeScript support throughout the codebase
- MIT License for open source usage

### Changed
- Improved button and card hover animations
- Enhanced focus states for better accessibility
- Improved keyboard navigation in dashboard
- Better scrollbar styling
- Refined visual hierarchy and spacing

### Fixed
- Keyboard accessibility issues in quick setup flow
- Focus state management in skills gallery
- ChatInput placeholder text handling
- Chat component rendering consistency

## [0.1.0] - Initial Development

### Added
- Project scaffolding and structure
- Basic extension manifest configuration
- Core architectural setup
- Background and content script foundation
